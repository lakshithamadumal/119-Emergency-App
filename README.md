# 119 - Emergency Response & Safety App 🚑🚨

**119** is a high-performance, modern Android application designed to provide instantaneous help during emergencies. Built with a focus on speed, reliability, and user-centric design, it allows users to trigger SOS alerts and contact emergency services with a single tap.

---

## ✨ Key Features

* **Modern UI/UX**
  Premium purple gradient interface with Material Design components and pixel-perfect layouts inspired by modern Figma designs.

* **One-Tap SOS Alert**
  A prominent SOS button designed to trigger emergency reporting workflows instantly.

* **Realtime Active Reports Dashboard**
  Displays live emergency report statistics using Firebase Realtime Database synchronization.

* **Firebase Authentication System**
  Secure user registration, login, and session handling using Email/Password authentication.

* **Cloud + Local Data Sync**
  Firebase Realtime Database handles live data while **SQLite LocalDB** provides offline caching and faster data access.

* **Admin Web Dashboard**
  A browser-based dashboard built with **PHP, MySQL, HTML, CSS, and JavaScript** for monitoring users, reports, and system activity.

* **REST-style API Integration**
  PHP endpoints connect the Android application, Firebase data flow, and MySQL dashboard.

* **Advanced Navigation Animations**
  Smooth screen transitions using slide and fade animations for a premium mobile experience.

* **Password Recovery**
  Fully functional "Forgot Password" workflow using Firebase email recovery.

* **Responsive ConstraintLayout UI**
  XML-based layouts optimized for different Android screen sizes.

## 🛠️ Tech Stack

### Mobile Application
- **Language:** Java
- **IDE:** Android Studio
- **UI Layout:** XML + ConstraintLayout
- **Design:** Material Design Components + Custom Drawables

### Backend & Cloud
- **Firebase Authentication**
- **Firebase Realtime Database**
- **PHP REST API**

### Local Storage
- **SQLite (LocalDB)**  
Used for offline caching and temporary data storage.

### Web Dashboard
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL

### Build System
- **Gradle (Groovy DSL)**
